##  Features

 User Authentication (Login & Signup using JWT + bcrypt)  
 Create and manage multiple boards  
 Add, update, and delete todos within boards  
 Track todo status (`todo`, `in_progress`, `done`)  
 Assign priorities (`Low`, `Medium`, `High`) 
 Clean, responsive UI built with Tailwind CSS  
 REST API backend with MongoDB integration

---

##  Tech Stack

### Frontend:
- React (Vite)
- React Router DOM
- Axios
- Tailwind CSS

### Backend:
- Node.js
- Express.js
- MongoDB + Mongoose
- bcryptjs (for password hashing)
- dotenv + cors
